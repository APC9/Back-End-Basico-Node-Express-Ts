# WebServer - RestServer

Ejecutar el comando :

```
  npm install
```
para reconstruir los modulos node


Ejecutar el compilador de TypeScript :

```
  tsc --watch
```

Levantar el servidor  :

```
  nodemon dist/app.js
```


Documentacion de los End-Points:
```
